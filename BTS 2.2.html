Cosa serve bootstrap?

Bootstrap è un framework front-end gratuito per uno sviluppo Web più rapido e semplice

Bootstrap include modelli di progettazione basati su HTML e CSS per tipografia, moduli, pulsanti, tabelle, navigazione, modali, caroselli di immagini e molti altri, nonché plug-in JavaScript opzionali Bootstrap ti dà anche la possibilità di creare facilmente progetti reattivi

Cosa devi importare per creare una pagina web con bootstrap?

<link rel="stylesheet" href="URL">

Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

molti utenti hanno già scaricato Bootstrap 4 da MaxCDN quando visitano un altro sito. Di conseguenza, verrà caricato dalla cache quando visitano il tuo sito, il che porta a tempi di caricamento più rapidi. Inoltre, la maggior parte dei CDN si assicurerà che una volta che un utente richiede un file da esso, verrà servito dal server più vicino a loro, il che porta anche a tempi di caricamento più rapidi.

A cosa serve questo attributo del tag meta? content="width=device-width, initial-scale=1

La width=device-widthparte imposta la larghezza della pagina per seguire la larghezza dello schermo del dispositivo (che varierà a seconda del dispositivo). La initial-scale=1parte imposta il livello di zoom iniziale quando la pagina viene caricata per la prima volta dal browser.

Che differenza c’è tra Container e Container Fluid (prova il container fluid creando una pagina chiamata BTS_1.1.1.html)

La .containerclasse fornisce un contenitore responsive a larghezza fissa La .container-fluidclasse fornisce un contenitore a larghezza intera , che copre l'intera larghezza della finestra
